# Lucky Grand Hotel – Hotel Management Web Application

Lucky Grand Hotel is a simple web-based hotel management and booking system built using HTML, CSS, and JavaScript. The application allows users to log in, book rooms, order food, generate bills, and download a PDF invoice. It also includes a basic admin dashboard with analytics.

Features --
* User login and signup system
* Room booking with check-in and check-out dates
* Food ordering system
* Payment simulation page
* Automatic bill generation
* Download bill as PDF using jsPDF
* Customer review section
* Admin dashboard with charts using Chart.js
* Image slider and room gallery

Technologies Used  -- 
* HTML5 – Website structure
* CSS3 – Styling and layout
* JavaScript – Application functionality
* Chart.js – Admin analytics charts
* jsPDF – PDF bill generation
